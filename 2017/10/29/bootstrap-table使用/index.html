bootstrap-table使用

发表于 2017-10-29

简介

bootstrap-table 是基于 Bootstrap 的 jQuery 表格插件，通过简单的设置，就可以拥有强大的单选、多选、排序、分页，以及编辑、导出、过滤（扩展）等等的功能。

文档地址 http://bootstrap-table.wenzhixin.net.cn/zh-cn/

主要功能

  • 支持 Bootstrap 3 和 Bootstrap 2
  • 自适应界面
  • 固定表头
  • 非常丰富的配置参数
  • 直接通过标签使用
  • 显示/隐藏列
  • 显示/隐藏表头
  • 通过 AJAX 获取 JSON 格式的数据
  • 支持排序
  • 格式化表格
  • 支持单选或者多选
  • 强大的分页功能
  • 支持卡片视图
  • 支持多语言
  • 支持插件

基本使用

引入

启用 Bootstrap Table 插件

方式一：通过 data 属性的方式

对普通的 table 设置 data-toggle="table"，设置远程的 url 如 data-url="data1.json" 来加载数据

方式二：通过 JavaScript 的方式

通过表格 id 来启用 bootstrap table。设置远程的 url 如 url: 'data1.json' 或者data数组data: [{name: '张三'}]来加载数据，设置columns数组columns:[{field: 'name',title: '名字'}设置表格内容。columns可以设置thead>th的data-field替代。

分页

bootrap-table分页有两种方式。服务端分页和客户端分页。

服务端分页

设置sidePagination: "server"，必须设置 服务器数据地址url: queryUrl或者重写ajax方法。

客户端分页

设置sidePagination: "client"，可以设置url或者data方式获取数据，一次加载所有数据，由客户端分页。data格式[{},{},{}]

自定义单元格内容和操作

当单元格不仅仅是需要数据，还需要样式内容或操作时，例如：

可以配合使用formatter和events两个配置。

formatter

columns中的对象的配置参数，是个方法，返回内容是字符串，可以返回拼接成html的字符串来定义单元格的样式。
方法包含三个参数：
value：当前单元格的字段数值
row：当前行的数据
index：当前行的下标索引

events

当使用了formatter时，可以配合events设置表格的监听事件操作。是一个对象，通过属性'事件类型 选择器':function(e, value, row, index)，定义一个事件。
事件方法，包含三个参数：
event: jquery的事件
value: 当前单元格的数据值
row: 当前行的数据
index: 当前行的下标索引

表格的事件

表格事件可以通过初始化$('#mytable').bootstrapTable({options})时配置，也可以通过$('#mytable').on('type',function)设置。

国际化、多语言

使用方式

在引入boostrap-table.js后引入本地配置的多语言文件bootstrap-table-zh-CN.js，然后根据需求切换不同语言配置文件

多语言配置参数

配置的都是一个方法，返回字符串

  • formatLoadingMessage
    正在加载数据时的提示，默认值”Loading, please wait…”

  • formatRecordsPerPage
    记录条数，默认值”%s records per page”
    参数pageNumber（条数）

  • formatShowingRows
    当前页的记录信息，默认值’Showing %s to %s of %s rows’
    参数pageFrom（记录开始索引）, pageTo（记录结束索引）, totalRows（总记录数）

  • formatDetailPagination
    详细记录条数，默认值’Showing %s rows’
    参数totalRows（总记录数）

  • formatNoMatches
    没有数据时提示信息，默认值’No matching records found’

  • formatSearch
    搜索提示，默认值’Search’

  • formatRefresh
    刷新提示，默认值’Refresh’

  • formatToggle
    切换提示，默认值’Toggle’

  • formatColumns
    列提示，默认值’Columns’

  • formatAllRows
    所有列提示，默认值’All’
